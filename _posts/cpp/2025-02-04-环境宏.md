## 操作系统

| 操作系统     | 预定义宏                     |
| ------------ | ---------------------------- |
| Windows      | `_WIN32`, `_WIN64`           |
| macOS        | `__APPLE__`, `__MACH__`      |
| Linux        | `__linux__`, `__gnu_linux__` |
| Unix         | `__unix__`, `__unix`         |
| Android      | `__ANDROID__`                |

## 编译器

| 编译器 | 预定义宏                     |
| ------ | ---------------------------- |
| GCC    | `__GNUC__`, `__GNUG__`       |
| Clang  | `__clang__`                  |
| MSVC   | `_MSC_VER`                   |
| MinGW  | `__MINGW32__`, `__MINGW64__` |
