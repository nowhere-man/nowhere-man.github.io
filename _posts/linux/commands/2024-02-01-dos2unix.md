### 修改单个文件

```bash
dos2unix filename
```

### 修改整个目录

```bash
find . -type f -exec dos2unix {} \;
```

# unix2dos

### 修改单个文件

```bash
unix2dos filename
```

### 修改整个目录

```bash
find . -type f -exec unix2dos {} \;
```
